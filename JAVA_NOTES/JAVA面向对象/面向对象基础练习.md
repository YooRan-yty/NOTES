# 1.继承

## 使用extends关键字实现继承

```java
class A
{
  public void a()
  {
    System.out.println("THIS IS A");
  }
}

class B extends A
{
  public void b()
  {
    System.out.println("THIS IS B");
  }
}

public class TEST
{
  public static void main(String[] args)
  {
    B myb = new B;
    myb.a();  //此时B已经继承了A的所有属性，可通过B来访问A中的方法：a()
    myb.b();  //继承A的同时，保留了自己的属性和方法，也可以访问自身的方法:b()
  }
}
```



# 2.多态

## 一个任务，多种不同的执行方式

```java
//Shape.java
public interface Shape  //创建名为Shape的接口
{
  public void draw();
}
```



```java
//Circle.java
public class Circle implements Shape
{
  public void draw()
  {
    System.out.println("CIRCLE");
  }
}
```



```java
//Square.java
public class Square implements Shape
{
  public void draw()
  {
    System.out.println("SQUARE");
  }
}
```



```java
//test.java
public class test
{
  public static void main(String[] arg)
  {
    Shape sh1 = new Circle();
    sh1.draw();
    
    Shape sh2 = new Square();
    sh2.draw();
  }
}
```
