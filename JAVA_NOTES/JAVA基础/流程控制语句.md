## 2.分支结构

### 2.1 if语句

#### 2.1.1第一种格式 if

```java
格式：
  if (判断条件){    //判断条件为true则执行语句；
    语句；
  }
```



#### 2.1.2 第二种格式 if..else

```java
格式：
  if (判断条件){   
    语句；  //为true执行；
  } else {
    语句； //为false执行；
  }
```



#### 2.1.3 第三种格式 if..else if ..else

```java
格式：
  if(判断条件1){
    语句体1；
  } else if(判断条件2){
    语句体2；
  } 
....
  else{
    语句体n;
  }
```
